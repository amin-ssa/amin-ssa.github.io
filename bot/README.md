# 🤖 بوت إدارة الطلبات والتنبيهات التلقائي 24/7

هذا البوت يقوم بمراقبة طلبات موقع **Hicham Y Saif** على قاعدة بيانات Firebase Firestore فورياً، ويرسل إشعارات بريدية وتنبيهات تلغرام تفاعلية تتيح لك قبول أو رفض الطلبات بنقرة واحدة، بالإضافة إلى محاكاة إرسال المتابعين التلقائي.

---

## 🚀 كيفية تشغيل البوت ليعمل 24/7 (حتى لو كان جهازك مطفأً)

لجعل البوت يعمل بدون توقف، يفضل رفعه على منصة سحابية مجانية مثل **Render** أو **Koyeb**.

### الخطوة 1: الحصول على ملف Firebase Credentials
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/).
2. اختر مشروعك (`hicham-y-saif`).
3. اضغط على ترس الإعدادات ⚙️ ثم اختر **Project settings**.
4. انتقل إلى تبويب **Service accounts**.
5. اضغط على زر **Generate new private key** وقم بتحميل الملف.
6. قم بتغيير اسم الملف إلى `serviceAccountKey.json` وضعه داخل مجلد `bot/` بجانب ملف `index.js`.

### الخطوة 2: تهيئة ملف الإعدادات البيئية (`.env`)
قم بإنشاء ملف باسم `.env` (أو عدل الملف الحالي) واملأ البيانات التالية:

---

### الخطوة 3: الرفع والتشغيل 24/7 (مجاناً)

#### الخيار أ: منصة Render.com (موصى به)
1. قم بإنشاء حساب مجاني على [Render](https://render.com/).
2. اضغط على **New** ثم اختر **Background Service** (أو Web Service).
3. اربط مستودع GitHub الخاص بك.
4. حدد المجلد الفرعي كـ `bot/` أو ارفع مجلد البوت بشكل مستقل.
5. في قسم **Environment Variables**، أضف المتغيرات الموجودة بملف `.env`.
6. سيقوم Render بتشغيل البوت تلقائياً ومراقبته 24/7 مجاناً!
